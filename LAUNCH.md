# ChangelogAI — Launch Document

> **Mission:** Turn GitHub commits into beautiful release notes in 10 seconds, powered by AI.

## Product Summary

**ChangelogAI** is a micro-SaaS tool for indie developers and small teams who hate writing changelogs manually. Input a GitHub repo + version tag range → get a polished, user-friendly Markdown changelog instantly.

### MVP (v0.1)
- ✅ Web UI: input repo + tags → generate changelog
- ✅ GitHub API integration (public + private repos)

## Deployment: Vercel

### Step 1: Create GitHub repo
```bash
cd products/P2-F/changelog-ai
git init
git add .
git commit -m "feat: initial ChangelogAI MVP"
gh repo create changelog-ai --public --source=. --remote=origin --push
```

### Step 2: Deploy to Vercel
```bash
npx vercel --prod
```

Or connect via Vercel dashboard:
1. Go to vercel.com → New Project
2. Import GitHub repo
3. Framework: Next.js
4. Add environment variables (from `.env.example`)

### Step 3: Setup Stripe
1. Create account at stripe.com
2. Create a product: "ChangelogAI Pro" at $9/month
3. Copy Price ID → `STRIPE_PRICE_PRO`
4. Add webhook: `https://your-domain.vercel.app/api/stripe/webhook`
5. Events: `checkout.session.completed`, `customer.subscription.deleted`

### Step 4: Configure env vars in Vercel
```
ANTHROPIC_API_KEY=sk-ant-...
STRIPE_SECRET_KEY=sk_live_...
STRIPE_PUBLISHABLE_KEY=pk_live_...
STRIPE_WEBHOOK_SECRET=whsec_...
STRIPE_PRICE_PRO=price_...
NEXT_PUBLIC_APP_URL=https://your-domain.vercel.app
```

## CLI Usage

```bash
# Install
pip install anthropic requests
chmod +x cli/changelog-ai.py

# Generate from tag range
ANTHROPIC_API_KEY=sk-ant-... python3 cli/changelog-ai.py vercel/next.js --from v14.0.0 --to v14.1.0

# Save to file
python3 cli/changelog-ai.py facebook/react --to v18.3.0 --output CHANGELOG.md

# JSON format
python3 cli/changelog-ai.py my-org/repo --format json
```
