## What Is ChangelogAI?

**ChangelogAI** transforms GitHub commits and pull requests into polished, user-friendly release notes in seconds — powered by Claude AI.

**Target User:** Independent developers and small teams who ship software regularly but hate writing changelogs manually.

---

## MVP Feature Set

### ✅ Web Application
- **Landing page** (`/`) — Hero, features, social proof, pricing preview, CTA
- **Generator** (`/generate`) — Input GitHub repo + tag range → AI-generated changelog
- **Pricing page** (`/pricing`) — Free vs Pro comparison with FAQ
- **Checkout** (`/checkout`) — Stripe-powered upgrade flow with 7-day trial
- **Success page** (`/checkout/success`) — Post-payment confirmation

### ✅ AI-Powered Generation
- **Primary:** Claude claude-3-5-haiku-20241022 (cost-efficient, fast)
- **Fallback:** Rule-based categorization (works without API key)
- **Smart categorization:** 🚀 Features, 🐛 Bug Fixes, ⚠️ Breaking Changes, 🔒 Security, 📦 Deps, 🔧 Internal
- **Contributors list** auto-extracted from commits

### ✅ GitHub Integration
- Public repos — no token required
- Private repos — GitHub Personal Access Token (Pro feature)
- Tag comparison via GitHub Compare API
- PR reference extraction from commit messages

### ✅ Stripe Payments
- Free tier: 3 changelogs/month (IP-based rate limiting)
- Pro tier: $9/month, unlimited, private repo support
- 7-day free trial for Pro
- Webhook handling for subscription lifecycle

### ✅ CLI Tool
- `cli/changelog-ai.py` — Python script, no install required
- Supports `--from`, `--to`, `--output`, `--format` flags
- Calls local AI (Anthropic SDK) or ChangelogAI API
